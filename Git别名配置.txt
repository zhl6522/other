Windows下使用Git简写：
	~/.gitconfig下不能配置带有git简写的（比如：gst=git status）
	操作：
		1.新建 ~/.bashrc文件
		2.编辑文档
			alias ga='git add .'
			alias gb='git branch'
			alias gt='git tag'
			alias gba='git branch -a'
			alias gd='git diff'
			alias gst='git status'
			alias gcm='git commit -m'
			alias gco='git checkout'
			alias gm='git merge'
			alias glo='git pull origin'
			alias gpo='git push origin'
			alias gst='git status'
		3.保存并source ~/.bashrc即可
		
	alias可以查看Git别名配置
